# Changelog

All notable changes to this project will be documented in this file.

## [v1.1.0] - 2025-03-10

### Added
- **热门关键词功能**
  - 自动从模型标题中提取高频关键词（20个）
  - 支持屏蔽不感兴趣的关键词
  - 屏蔽后自动补位新关键词
  - 点击关键词可快速搜索相关模型
  - 显示已屏蔽关键词列表，支持取消屏蔽

### Changed
- **UI 优化**
  - 侧边栏磨砂玻璃效果（backdrop-filter: blur）
  - 下拉菜单磨砂玻璃效果
  - 关键词标签渐变背景和悬停动画
  - 移动端模型卡片标题显示优化

### Fixed
- 修复模型详情页主题切换不同步问题
- 修复移动端热门关键词折叠功能
- 修复桌面端侧边栏折叠时关键词不隐藏问题

## [v1.0.0] - Initial Release

### Added
- 模型采集与归档功能
- 离线页面生成
- 模型库浏览
- 缺失 3MF 重试
- 浏览器插件一键归档
- 浅色/暗黑主题切换
- 批量删除功能
- 模型相似度排序
- 响应式设计（桌面端/移动端）
- 侧边栏收缩/展开
- 5 线程并发下载
- 国内/国际平台 Cookie 分离
