### 4. 并发下载优化
**影响**: 显著加速模型归档速度  
**工作量**: 中等 (3-4天)

**关键改进**:
- 使用 `asyncio` + `aiohttp` 实现并发下载
- 可配置并发数,避免过载
- 添加进度条显示

**示例代码**:
```python
import asyncio
import aiohttp
from tqdm import tqdm

async def download_file_async(session, url, dest):
    async with session.get(url) as resp:
        resp.raise_for_status()
        dest.parent.mkdir(parents=True, exist_ok=True)
        with open(dest, "wb") as f:
            async for chunk in resp.content.iter_chunked(8192):
                f.write(chunk)

async def download_all_images(images, max_concurrent=5):
    async with aiohttp.ClientSession() as session:
        semaphore = asyncio.Semaphore(max_concurrent)
        async def bounded_download(img):
            async with semaphore:
                await download_file_async(session, img["url"], img["dest"])
        
        tasks = [bounded_download(img) for img in images]
        for coro in tqdm(asyncio.as_completed(tasks), total=len(tasks)):
            await coro
```

### 5. 前端优化
**影响**: 提升大数据量下的用户体验  
**工作量**: 中等 (2-3天)

**关键改进**:
- 后端分页,减少数据传输
- 前端虚拟滚动(可选)
- 搜索移到后端,支持模糊匹配

**后端分页**:
```python
@app.get("/api/gallery")
async def api_gallery(
    page: int = 1, 
    page_size: int = 20, 
    search: str = "",
    sort_by: str = "collected_at",
    order: str = "desc"
):
    items = scan_gallery(CFG)
    
    # 搜索
    if search:
        items = [i for i in items if search.lower() in i["title"].lower() or 
                 any(search.lower() in tag.lower() for tag in i["tags"])]
    
    # 排序
    reverse = (order == "desc")
    items.sort(key=lambda x: x.get(sort_by, ""), reverse=reverse)
    
    # 分页
    total = len(items)
    start = (page - 1) * page_size
    end = start + page_size
    
    return {
        "items": items[start:end],
        "total": total,
        "page": page,
        "page_size": page_size
    }
```

**影响**: 减少文件系统 I/O,提升响应速度  
**工作量**: 小 (1-2天)

**关键改进**:
- 缓存模型库扫描结果
- 支持手动刷新缓存
- 可选 Redis 支持分布式部署

**内存缓存**:
```python
from datetime import datetime, timedelta

_gallery_cache = {"data": None, "expires_at": None}

def scan_gallery_cached(cfg, ttl=300):
    now = datetime.now()
    if _gallery_cache["data"] and _gallery_cache["expires_at"] > now:
        logger.info("使用缓存数据")
        return _gallery_cache["data"]
    
    logger.info("重新扫描模型库")
    data = scan_gallery(cfg)
    _gallery_cache["data"] = data
    _gallery_cache["expires_at"] = now + timedelta(seconds=ttl)
    return data

@app.post("/api/gallery/refresh")
async def refresh_gallery():
    _gallery_cache["data"] = None
    return {"status": "ok", "message": "缓存已清除"}
```
