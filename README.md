# MakerWorld 本地归档小应用

![MakerWorld Archive](https://aliyun-wb-h9vflo19he.oss-cn-shanghai.aliyuncs.com/use/makerworld_archive.png)

一个用于归档 MakerWorld 模型到本地的项目，支持模型采集、离线页面生成、模型库浏览、缺失 3MF 重试，以及浏览器插件一键归档。

原版地址：https://github.com/sonicmingit/mw_archive_py

## 界面优化（自定义修改）

### 交互体验
- **模型卡片交互优化**：默认状态只显示封面和标题，悬停时平滑显示详细信息（点赞数、收藏数等），实现平滑过渡动画
- **自定义确认对话框**：替换默认的浏览器 confirm 对话框为美观的自定义模态对话框
- **响应式设计**：确保在不同屏幕尺寸下都能正常显示
- **多选删除功能**：支持批量选择多个模型进行删除操作
- **侧边栏优化**：添加收缩/展开按钮，支持状态持久化

### 视觉增强
- **归档失败提示**：当模型在 MakerWorld 归档失败（cookie 失效）时，在首页该模型卡片上显示红色提示框
- **提示框美化**：优化归档失败提示框样式，添加渐变背景、阴影和动画效果
- **浅色/暗黑模式切换**：支持一键切换主题，自动检测系统偏好，状态持久化保存

## 性能优化

### 并发下载
- 图片下载改为 5 线程并发，速度提升约 3-5 倍
- 3MF 文件支持多实例同时下载
- 智能任务调度，避免重复下载

### 国内/国际平台 Cookie 分离
- 支持分别配置国内版 和国际版 Cookie
- 归档时自动根据 URL 选择对应平台的 Cookie
- 未设置时自动尝试备用平台 Cookie

## 功能增强

### 删除模型自动清理
- 删除模型卡片时自动清理对应的缺失 3MF 记录
- 同步更新收藏/已打印标记

### 模型相似度排序
- 新增"模型相似度倒序"排序选项
- 基于 Levenshtein 编辑距离算法计算模型标题相似度
- 自动将相似模型分组排列，方便浏览相关模型
- 支持跨设计师的相似度比较（基于标题相似度）

## 与原版的区别

| 功能 | 原版 | 自定义版 |
|------|------|----------|
| 卡片显示 | 默认显示完整信息 | 默认简洁显示，悬停展开详情 |
| 确认对话框 | 浏览器默认对话框 | 美化的自定义模态对话框 |
| 归档失败提示 | 无 | 卡片上显示红色提示框 |
| 批量删除 | 无 | 支持多选批量删除 |
| 侧边栏 | 固定宽度 | 可收缩/展开 |
| 主题切换 | 无 | 浅色/暗黑模式切换 |
| 下载方式 | 单线程串行 | 5 线程并发 |
| 删除清理 | 手动清理缺失记录 | 自动清理 |
| 相似度排序 | 无 | 支持模型相似度倒序排序，自动分组相似模型 |
